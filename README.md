# Building

## Jailbreak

1. install [Theos](https://theos.dev/docs/installation)

2. ```bash
   make package FINALPACKAGE=1
   ```

## TrollStore

1. install [Theos](https://theos.dev/docs/installation)
2. ```bash
   make package FINALPACKAGE=1 TROLLSTORE=1
   ```

> If you are going to patch it, get the right version of  Wallet App for your Device.
